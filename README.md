# fixel

## Description

Randomly reorders the pixels of an image to create a new image.

## Usage

Step 1:

Clone repo.

Step 2:

```
cargo build --release
```

Step 3:

```
sudo cp target/release/fixel /usr/local/bin
```

Step 4:

```
fixel --help
```
